A local gym "Fitness Nepal" in Dharan needs a membership management system.
Create "gymmembernepal.java" class:
• Attributes: membername, membershiptype (Monthly, Quarterly, Yearly),
monthlyfee, joindate
• Constructor to initialize all attributes
• Methods:
calculateannualfee() – returns yearly cost with Nepali tax (13% VAT)
getdiscountedfee(double discountpercent) – returns discounted
monthly fee for students/seniors
iseligiblefordiscount() – returns true if member for >6 months (loyalty
discount)
displaymemberinfo() – shows formatted member details in Nepali style
• Test with different membership types: Basic (Rs. 1500), Premium (Rs. 2500),
VIP (Rs. 4000)


# Gym Membership Management System (Java)

##  Description

This is a simple Java console program that manages gym membership details.
The program allows users to enter their information and then calculates:

* Annual fee including 13% VAT
* Discounted fee (if eligible)
* Displays membership details in a structured format

This project demonstrates the use of:

* Classes and Objects
* Constructors
* Methods
* Conditional statements
* User input using Scanner

---

##  Features

* Accepts member details from the user
* Calculates annual fee with VAT
* Checks discount eligibility
* Displays membership information

---

## Technologies Used

* Java
* VS Code

---

##  How to Run

1. Open the project in VS Code
2. Compile the program:

```
javac GymMemberNepal.java
```

3. Run the program:

```
java GymMemberNepal
```
